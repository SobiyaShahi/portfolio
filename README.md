# Sobiya Shahi — Portfolio

A modern single-page portfolio website built with React, TypeScript, Vite, and Framer Motion.

## Sections

- **Home** — Engaging hero with animated introduction
- **About** — Personal story and highlights
- **Skills** — Technical & soft skills with animated progress bars
- **Projects** — Featured creative work
- **Experience** — Professional timeline
- **Education** — Academic journey
- **Certificates** — Tool proficiencies
- **Contact** — Get in touch form

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Framer Motion (animations)
- React Icons
